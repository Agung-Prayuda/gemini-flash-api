Catatan Penting

1. Untuk menjalankan aplikasi ChatbotAI klik index.html di dalam folder Public

2. Proyek ini tidak menyertakan Google Gemini API Key secara langsung di repository.  
   Hal ini dilakukan untuk alasan keamanan agar kredensial tidak terekspos publik.  

3. Jika Anda ingin menjalankan proyek ini secara lokal, silakan klik file `.env` dan masukkan API key Anda sendiri
